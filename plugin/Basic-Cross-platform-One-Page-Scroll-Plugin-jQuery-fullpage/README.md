# fullpage.js
jQuery轻量级的fullpage插件
内置thrrote节流函数，性能更稳定

# Demo
[DEMO](https://poezhang.github.io/fullpage/demo/index.html)

# Use
### HTML:
```
<div id="fullpage">
		<section>1</section>
		<section>2</section>
		<section>3</section>
		<section>4</section>
		<section>5</section>
		<section>6</section>
	</div>
```
### CSS:
```
#fullpage{
  overflow:hidden;
}
```
### JS:
```
$(function(){
  $('#fullpage').fullpage();
}
```
